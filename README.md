# Ollama Console

Projeto desenvolvido em .NET Core para aplicar o conhecimento e fazer com que a aplicaçao interaja com um modelo de LLM hospedado local.

Importante: Ter uma instancia do Ollama configurada e com um modelo de IA ja configurado para analisar os prompts.

## 🎯 Project Overview

**Ollama.Console** Neste projeto temos alguns iniciadores para testar cada passo do tutorial

- **Chat** — Simple para fazer uma pergunta ao modelo e obter uma resposta em formato texto.
- **Track Assets** — Monitor stocks, bonds, and other investment instruments
- **Record Trades** — Log buy/sell transactions and track trading history
- **View Portfolios** — Get a comprehensive view of all portfolios and asset allocation
- **Bank Integration** — Link and manage multiple banking sources
- **Account Administration** — User authentication and account management
